# Java 1 - Assignment 4

Java program to play Ricochet Robots

## How to Run

### In VS Code
Run `Main.java` using the run button.

### In Terminal
From the 'Assignment4' directory:

Compile: javac *.java

Run: java Main

## How to play

Each turn the asks for two things:

1. A **robot name** - e.g. 'AA', 'BB', 'CC' (case sensitive).
2. A **direction** - 'n', 's', 'e', 'w', or the full word.

The chosen robot slides in that direction until it hits a wall, another robot, or the edge of the the board. The game ends when any of the robos stops on the goal cell 'gg'. After winning you can choose to play again or quit.
