# Java 1 - Assignment 2

Java program to analyse text.

## How to Run

### In VS Code
Run `Assignment2.java` using the run button.

### In Terminal
Compile: javac Assignment2.java

Run: java Assignment2


##The program

Once the program runs, type the name of the text file: either "dengrimmeaelling.txt"

or "theuglyduckling.txt". The program must be run again to try the other text.
